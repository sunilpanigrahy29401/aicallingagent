How it Works

3. Click Start Call
4. LiveKit dispatches the agent job
5. The AI dials the number via Vobiz SIP
6. Gemini Live handles real-time audio
7. Call result appears in Call Logs

🔧 Setup Guide
Quick Start
  1. Go to Settings and enter your LiveKit API credentials.
  2. Enter your Gemini API key (Free tier is fine).
  3. Enter your Vobiz SIP credentials and click Create Trunk.
  4. Go to AI Prompt and customize it for your business.
  5. Go to Single Call and test with your own phone number.
Note: You don't need Deepgram STT or separate TTS if using Gemini Live (Realtime Mode = True). Gemini Live processes audio directly.
